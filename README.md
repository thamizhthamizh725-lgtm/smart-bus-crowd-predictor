# 🚌 Smart Bus Crowd & Delay Prediction System

## 🚀 Overview
This project predicts bus crowd levels based on delay data to help commuters plan their travel efficiently.

## 🎯 Problem Statement
Public transport users often face uncertainty in crowd levels and delays. This system provides insights and predictions.

## 🧠 Features
- Passenger trend analysis
- Crowd prediction using Machine Learning
- Interactive dashboard (Streamlit)

## 🛠️ Technologies Used
- Python
- Pandas
- Streamlit
- Scikit-learn
- Matplotlib

## 📊 Input
- Delay in minutes

## 📈 Output
- Predicted passenger count
- Crowd level (Low / Medium / High)

## ▶️ How to Run
```bash
pip install pandas streamlit scikit-learn matplotlib
streamlit run app.py
